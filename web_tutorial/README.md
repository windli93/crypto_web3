# Web3_Tutorial

Web3科学家极简入门指南

目标：学习并使用 Web3.py 模块实现链上数据查询、转账、合约交互等简单功能


## 00: 前期准备工作

1. 安装 Python3
2. 安装 web3.py 库 `pip install web3`
3. 申请 Infura API Key: https://infura.io/
